
문자 단위 출력 스트림 

** Writer ** 

- 문자 단위 출력 스트림의 최상위 추상 클래스 
- 많은 추상 메서드가 선언되어 있고 이를 하위 스트림이 상속 받아 구현 되어 있다. 
(문서 권장) 

-- 주요 하위 클래스 
FileWriter : 파일에다가 문자 단위로 출력하는 스트림 클래스 
OutputStreamWriter : 바이트 단위의 자료를 문자로 변환해 출력해주는 보조 스트림 
BufferedWriter : 문자로 쓸 때 가변 배열을 제공하여 한꺼번에 쓸 수 있는 보조 스트림
